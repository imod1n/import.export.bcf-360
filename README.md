# "Импорт и экспорт BCF"

Плагин для платформы **Topomatic 360 / Albatros**.

## Возможности

Просмотр, создание и редактирование замечаний в формате BCF

# Установка зависимостей

```sh
npm install
```

# Сборка

```sh
npm run build
```

# Отладка

```sh
npm run serve
```
